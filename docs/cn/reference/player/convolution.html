应用卷积效应

ConvolutionFilter

你可以应用矩阵卷积滤波器效果。 卷积通过将输入图像中的像素与相邻的像素结合在一起来创建一个图像。 卷积可以用来实现广泛的图像效果，如模糊、边缘检测、锐化、浮雕和倒角。

  • matrixX

    矩阵的X维

  • matrixY

    矩阵的Y维

  • matrix

    一个用于矩阵转换的数值数组

  • divisor

    矩阵转换时使用的除数

  • bias

    在矩阵转换结果中加入的偏差量

  • preserveAlpha

    是否保留阿尔法通道而不加滤镜效果，或者是否将卷积滤镜应用于阿尔法通道和颜色通道

  • clamp

    你是否需要夹紧图像

  • color

    用颜色替换源图像以外的像素

  • alpha

    颜色的阿尔法透明度值
