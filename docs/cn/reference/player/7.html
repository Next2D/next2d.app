对父级DisplayObjectContainer和根级DisplayObjectContainer的引用

parent

父属性指向父DisplayObjectContainer。 你可以使用多个父母来回溯到一个祖先。 但是，如果没有注册DisplayObjectContainer的子元素，将返回null。

root

根属性指向层次结构中最高级别的父级DisplayObjectContainer。 在大多数情况下，这将是舞台正下方的DisplayObjectContainer。 然而，如果没有注册的DisplayObjectContainer的孩子，它将返回空。
