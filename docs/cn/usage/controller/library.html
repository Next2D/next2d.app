库

你可以导入图片（jpeg、gif、png）、音频（mp3）、视频（mp4）和SWF文件作为外部数据。 你还可以管理在屏幕上创建的矢量图、文本和电影剪辑。

Next2D

Next2D 源于一个 JavaScript Flash 播放器项目 "swf2js"，旨在提供一个在任何设备上都可以运行的、最快的 2D 引擎。
Created by Toshiyuki Ienaga.
