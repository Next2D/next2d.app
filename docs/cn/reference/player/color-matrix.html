应用颜色矩阵效果

ColorMatrixFilter

通过对输入图像中每个像素的RGBA颜色和阿尔法值进行4×5的矩阵变换，可以创建一个由新的RGBA颜色和阿尔法值组成的结果。 这允许使用各种效果，包括饱和度变化、色调旋转和亮度α转换。

  • matrix

    4 x 5阵列用于颜色转换
