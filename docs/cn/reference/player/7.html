对父级 DisplayObjectContainer 和根级 DisplayObjectContainer 的引用

parent

父属性指向父 DisplayObjectContainer。你可以使用多个父母来回溯到一个祖先。但是，如果没有注册 DisplayObjectContainer 的子元素，将返回null。

root

根属性指向层次结构中最高级别的父级 DisplayObjectContainer。在大多数情况下，这将是Stage正下方的 DisplayObjectContainer。然而，如果没有注册的 DisplayObjectContainer 的孩子，它将返回空。
