DisplayList、DisplayObject 和 DisplayObjectContainer

DisplayList

DisplayList 由三个主要元素组成。Stage、DisplayObjectContainer 和 DisplayObject。

Stage

Stage 是一个容纳所有 DisplayObject 的对象。

DisplayObjectContainer

DisplayObjectContainer 是一个 DisplayObject，可以用来放置 DisplayObjects，它继承于以下四个类。

  • Loader
  • MovieClip
  • Sprite
  • Stage

DisplayObject

DisplayObject 是一个执行绘图过程的对象。从 DisplayObject 继承的9个类如下。

  • DisplayObjectContainer
  • Loader
  • MovieClip
  • Sprite
  • Stage
  • InteractiveObject
  • Shape
  • TextField
  • Video

Adding Objects

要将一个 DisplayObject 添加到一个 DisplayObjectContainer 中，请使用 DisplayObjectContainer 的 addChild 方法。
